## 📋 What is Tiny-Lab?

Tiny-Lab is a small machine learning research tool built for Apple Silicon computers. It helps you run machine learning experiments with a real control setup. It includes one built-in training path and tools for evaluating models you create. You don’t need to know programming to use it, and it runs efficiently on your Mac with Apple’s M1 or M2 chips.

If you are curious about exploring machine learning in a simplified environment, Tiny-Lab offers an easy start. It handles key steps automatically, so you can focus on your results.

## 🖥️ System Requirements

Before installing, make sure your computer meets these needs:

- **Operating System:** macOS 11 (Big Sur) or later  
- **Processor:** Apple Silicon (M1, M1 Pro, M1 Max, M2, or newer)  
- **Memory:** At least 8 GB RAM  
- **Storage:** 2 GB of free disk space  
- **Internet:** Required for downloading software and updates  

Tiny-Lab is designed specifically for Apple Silicon hardware. It will not run on Intel Macs or Windows computers.

## 🛠️ Using Tiny-Lab: Basic Controls

Once Tiny-Lab runs, you will see a simple interface:
